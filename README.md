# AppTimer
AppTimer is an windows dekstop app that shows you how long each of your apps has been running.

## Features
- See all open apps with how long they have been running
- App icons next to each app
- Shows each app start time
- Search bar to find specific apps
- Sort by name, time or start time
- Toggle between showing all system processes/files and only apps
- Dark and light modes
- Time updates every 3 seconds

  ## How to use
  1. Download AppTimer.exe from Releases
  2. Run it
  3. Your running apps will appear automatically
 
  ## Tech stack
  - Python 3.14
  - psutil
  - pywin32
  - tkinter
  - Pillow
  - auto-py-to-exe
